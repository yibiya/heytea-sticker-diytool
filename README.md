# 喜茶喜贴 DIY 工具 (Auto Painter)

这是一个使用 Python 制作的自动屏幕绘图工具，拥有一个现代化的图形用户界面。它可以加载一张图片，将其转换为线条轮廓，然后模拟鼠标操作在屏幕的指定区域绘制出来。

## ✨ 功能特性

- **🖼️ 现代 UI**: 使用 `customtkinter` 构建，界面美观，支持深色模式。
- **⚙️ 实时参数调整**:
    - **边缘细节**: 控制边缘检测的阈值，数值越小，细节越多。
    - **忽略短线**: 忽略指定长度以下的短小线条。
    - **自动连接**: 自动连接距离较近的线段，使轮廓更完整。
    - **笔画间隔**: 调整每次下笔之间的延迟，以适应不同性能的设备。
- **🎯 精准区域选择**:
    - 按下 `F8` 或点击 "锁定区域" 按钮，即可在屏幕上任意框选一个矩形区域作为绘图的目标画布。
- **▶️/⏹️ 快捷键控制**:
    - `F8`: 选择绘图区域。
    - `F9`: 在选定区域内开始绘图。
    - `F10`: 立即停止绘图。
- **📱 Scrcpy 投屏支持**:
    - 内置 `scrcpy` 启动功能。打开开关后，程序会自动启动 `scrcpy` 窗口，方便在安卓手机/平板上进行绘制。
    - **注意**: 需要将 `scrcpy` 工具包解压到程序目录下的 `scrcpy` 文件夹内。

## 🚀 如何使用

1.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **运行程序**:
    ```bash
    python draw.py
    ```

3.  **操作步骤**:
    1.  点击 **"1. 加载素材"** 按钮，选择一张你想要绘制的图片。
    2.  程序会自动对图片进行预处理，并在右侧显示轮廓预览图。
    3.  通过拖动左侧的滑块调整参数，直到预览效果满意为止。
    4.  按下 `F8` 键（或点击 **"2. 锁定区域"** 按钮），鼠标会变成十字形。按住左键在屏幕上拖动，框选出你想要绘图的区域（例如，喜茶小程序的画布）。
    5.  按下 `F9` 键（或点击 **"开始"** 按钮），程序将开始自动绘制。
    6.  如需中途停止，按下 `F10` 键（或点击 **"停止"** 按钮）。
