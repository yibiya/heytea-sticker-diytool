# 喜茶喜贴DIY

这是一个基于 Python 的小工具，可以帮你自动绘制“喜茶”风格的像素画或贴纸。你只需要提供一张图片，框选出想要绘制的区域，程序就会模拟鼠标操作，将图片的轮廓绘制出来。

## 功能

*   **图片处理**：使用 OpenCV (Canny 边缘检测) 提取图片的主要轮廓。
*   **参数调整**：可以实时调整边缘细节、忽略短线长度、连接断线距离等参数，以达到最佳效果。
*   **屏幕框选**：可以方便地在屏幕上选择任何区域作为你的“画板”。
*   **自动绘制**：使用`pyautogui`模拟鼠标点击和移动，自动完成绘图过程。
*   **全局热键**：使用 `F9` 开始绘制，`F10` 随时停止。

## 安装

1.  确保你的电脑已经安装了 Python。
2.  克隆或下载本项目。
3.  在项目目录下，通过命令行安装所需依赖：

    ```bash
    pip install -r requirements.txt
    ```

## 使用方法

1.  运行 `draw.py` 文件：
    ```bash
    python draw.py
    ```
2.  在弹出的程序窗口中，点击 **“1. 上传图片”** 按钮，选择你想要绘制的图片。
3.  程序会自动处理图片，并在窗口中显示黑白的线条预览图。你可以通过拖动上方的滑块来调整绘图效果。
4.  点击 **“2. 框选区域”** 按钮。此时屏幕会变暗，用鼠标左键拖拽出一个矩形区域，这个区域就是你希望程序进行绘制的地方（例如，某个画图软件的画布）。
5.  切换到你的画图软件（如系统自带的画图、PhotoShop 等），确保你的画笔已经准备好。
6.  按下 **F9** 键，程序将开始自动绘制。
7.  如果想要中途停止，可以随时按下 **F10** 键。

## 注意事项

*   程序开始绘制后，鼠标和键盘会被接管，请不要随意移动鼠标，以免影响绘制。
*   如果程序运行出错，可以将鼠标移动到屏幕的左上角，会触发 `pyautogui` 的 `FAILSAFE` 机制，强制停止程序。
*   绘制速度可以调节，如果太快导致线条不连贯，可以适当增加“笔画间隔”。
